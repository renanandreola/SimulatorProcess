Para iniciar, você sabe o que é um processo?

Os processos representam tarefas em execução, mas nem todas têm relação direta com algum aplicativo. Muitas delas são executadas em segundo plano.


Processo é uma entidade abstrata (uma estrutura no sistema operacional) que representa um software em execução. O sistema operacional atribui um identificador a cada processo, bem como descritores de arquivos, espaço de memória, prioridade, entre outras informações. Um processo pode possuir mais de um fluxo de execução (thread).




O que é um gerenciador?

O Gerenciador de Tarefas é uma das ferramentas nativas do Windows, sistema operacional da Microsoft. Com o recurso é possível visualizar todos os programas que estão abertos e rodando em segundo plano no PC, fechar aplicativos que travaram de forma rápida e ainda controlar o desempenho do seu computador. No Windows 10, o acesso é rápido pela barra de tarefas.



Você sabe o que é CPU?

CPU é a sigla para Central Process Unit, ou Unidade Central de Processamento.
